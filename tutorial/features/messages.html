💬 Messages

SHOT 関係者 thread (3 名以上) と 1対1 DM (2 名) を タブで振り分け表示。新規 DM は 29 名から選択可。未読/既読は localStorage で永続化。

📸 画面

✨ 主な機能

  • 🎬 SHOT タブ — review/QC 提出時に自動作成された SHOT 関係者 thread (PM/Director/Lead/担当者) を一覧表示。
  • 👤 DM タブ — 1 対 1 の DM を一覧表示。タブ名に未読数 (例: DM (未読 3)) 表示。
  • 📢 通知タブ — システム通知。
  • 未読 dot — 各 thread row に 🔴 dot + 強調背景 で 未読 thread を視覚化。
  • 新規 DM 作成 — ✏️ ボタン → モーダルで Calendar 全 user (自分以外 29 名) から相手選択。
  • thread 本文 → 緑 QC ビューアボタン — review 依頼本文中の URL は緑ボタン化され、click 一発で対象 shot の QC viewer へ遷移。

🛠 操作方法

  1. sidemenu 💬 Messages → メッセージ画面 (left thread 一覧 + right 本文)。
  2. thread row click → 右ペインに最新メッセージ表示 + 既読化 + sidemenu 赤丸も即時更新。
  3. 新規 DM ✏️ → 相手選択 → /messages?thread={new_id} に遷移。
  4. メッセージ入力欄 → 送信 → Push 通知 + SSE banner で全 SHOT 関係者に届く。
