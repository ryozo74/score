📅 Calendar

個人 / チーム / プロジェクト 別カレンダー。予定の追加・編集・参加者選択・ミーティングリンク設定が可能。

📸 画面

✨ 主な機能

  • 個人 / チーム / プロジェクト カレンダー — タブ切替で view 変更。
  • 予定の追加・編集 — タイトル / 日時 / 場所 / 参加者 / ミーティングリンク / 説明文。
  • 参加者選択 — Calendar API の user 一覧から選択。複数選択可。
  • ミーティングリンク — Zoom/Meet 等の URL を直接登録。ダッシュボードに緑ボタンで反映される。
  • イベント連動 — review/QC 依頼の SHOT に紐付けて関連 task 自動 link。

🛠 操作方法

  1. sidemenu 📅 Calendar → カレンダー画面。
  2. 日付セル click → 新規イベント作成 dialog。
  3. 既存イベント click → 編集 modal。
  4. 参加者列に複数 user を選択 → ダッシュボードに反映。
