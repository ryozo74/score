👤 Profile

プロフィール情報 (表示名・連絡先・avatar 画像) の管理。スタジオ内の全通知・コメントに表示される情報を編集。

📸 画面

✨ 主な機能

  • 表示名 / 連絡先 — Score 内で表示される情報。
  • avatar 画像 — PNG/JPG 形式で upload (max 5 MB)。Calendar に同期。
  • 所属 role 表示 — admin / lead / director / pm / user のいずれか。
  • 退勤統計 — 直近の退勤報告履歴 サマリ。

🛠 操作方法

  1. sidemenu 右下の自分の avatar click → /profile に遷移。
  2. avatar 画像欄 click → ファイル選択 → upload (Calendar API 経由)。
  3. 保存 → 即座に全画面で反映。
