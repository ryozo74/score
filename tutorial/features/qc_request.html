🔍 QC 依頼

Compositor が QC 依頼を提出 → SHOT 関係者 thread に自動投稿 + Director 直接 Approve/Retake 判定段階。部内 OK 前提で Director に上申。

📸 画面

✨ 主な機能

  • 提出種別 = QC — 部内 OK 済前提・Director 判定段階
  • Director 自動補完 — project.director が設定有 → 自動 participants 追加
  • Director 不在 enforce — Director 設定無 + mention 無 → HTTP 400 拒否 + 「mention で代替担当 選択」警告
  • PM 必須含む — QC = Director / Review = mention 主体 — どちらも PM 必須 (殿御方針)
  • task status → review — Calendar enum 'review' 自動 patch
  • Push + SSE 配信 — 通知到達 → 担当者 メンション気づき → msg 確認 → QC ビューアへ

🛠 操作方法

  1. /task/{task_id} で対象 asset card → 📤 既存 asset で QC 依頼 → modal で「🔍 QC」選択
  2. (Director 設定有 project) mention 不要・直接送信
  3. (Director 設定無 project) ⚠️ 警告 → mention で送信先 1 名以上選択
  4. 受信側: 通知 toast → SHOT thread → ▶ QC ビューアで確認 → Director 判定 (Approve/Retake)
