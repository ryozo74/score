⚙️ 通知設定

通知の配信経路 (📣 Web Push / 🟢 SSE / 🔴 画面 badge) と 通知種別 (Review / QC / DM / SHOT thread / @ mention) を個別に ON/OFF 設定。

📸 画面

✨ 主な機能

  • 📣 Web Push (OS 通知) — PC / モバイル の OS toast。離席中でも通知。要 ブラウザ通知許可 (1 回のみ)。
  • 🟢 在席中 banner (SSE) — Score を開いている間 画面右下に 紫アクセント白 toast。OS 通知許可不要。8 秒で消失。
  • 🔴 sidemenu 赤丸 + 通知センター — 常時 ON (設定で OFF 不可)。見落とし防止。
  • 通知種別 5 区分 — 📌 Review 依頼 / 🔍 QC 依頼 / 💬 DM 受信 / 👥 SHOT thread 投稿 / @ mention 含む の個別 ON/OFF。
  • 初回ログイン onboarding — Notification.permission=default の user に紫色 banner で 1 click 許可案内。
  • silent re-subscribe — 既許可 user は 起動時に subscription を自動 refresh。

🛠 操作方法

  1. sidemenu ⚙️ 通知設定 → 設定画面。
  2. 経路 / 種別 を checkbox で ON/OFF → 💾 保存 → 即時反映。
  3. 📣 Web Push を有効化 ボタン → 許可取得 + 購読登録。
  4. 🧪 テスト送信 → Push + SSE 両方発火 → 結果確認。
