🌐 Lang (言語切替)

UI 言語を 日本語 / 繁體中文 / English に切替。設定は localStorage に永続化。

📸 画面

✨ 主な機能

  • 3 言語対応 — 日本語 (default) / 繁體中文 / English。
  • 段階的拡張中 — 一部 文言は未翻訳 (順次対応)。
  • 固有名詞は translate="no" で 自動翻訳から除外。

🛠 操作方法

  1. sidemenu 🌐 Lang → 言語選択 modal。
  2. 言語選択 → 即座に UI 文言切替 (page reload なし)。
