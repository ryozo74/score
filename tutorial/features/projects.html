🎞️ Project

スタジオが受注した全プロジェクトの一覧と進捗状況を確認。各プロジェクトの shot 数・QC 通過率・リテイク発生数等を俯瞰可能。

📸 画面

✨ 主な機能

  • 全プロジェクト一覧 — オンライン (進行中) / オフライン (完了) / アーカイブ済 の 3 区分。
  • 進捗バー — 各プロジェクトの全 shot に対する完了率 (Approved 数 ÷ 全 shot 数)。
  • QC 通過率 / リテイク率 — 品質管理の健全性指標。
  • 担当チーム表示 — PM / Director / Lead 各役のアバター。
  • プロジェクト click → 詳細画面 — 全 shot 一覧 / 担当割当 / カレンダー 等。

🛠 操作方法

  1. sidemenu 🎞 Project → 全プロジェクト一覧。
  2. プロジェクトカード click → /project_detail/{id} で全 shot を確認。
  3. ステータスフィルタ (オンライン/オフライン/アーカイブ) で表示切替。

🗂 階層構造 — Project / SEQ / SHOT / Task

Score では制作対象を 4 階層 で管理します。各層が上位を含む親子関係。QC/Review 依頼の通知タイトルにもこの階層が反映されます (例: marukome / SEQ001 / SHOT_001 / Composite)。

📁

Project (案件・最上位)

スタジオが受注した 1 つの案件単位。クライアント・予算・納期・担当チーム (PM / Director / 各分野 Lead) が紐付く。

  • 例: marukome (味噌 CM 案件) / Bali (映画予告 案件)
  • 1 Project = 複数 SEQ + Production Tracker + 共有カレンダー + 担当割当
  • ステータス: オンライン (進行中) / オフライン (完了) / アーカイブ済 (圧縮保管)
🎞️

SEQ (シーケンス・場面単位)

Project 内の連続する場面 (シーケンス) のまとまり。物語の章単位・ロケーション単位・尺単位 等で区切る。

  • 例: SEQ001 (オープニング場面) / SEQ002 (本編 食卓場面)
  • 1 SEQ = 複数 SHOT のまとまり (通常 5〜50 SHOT)
  • 命名規則: SEQ + 3 桁数字 (例 SEQ001 / SEQ045)
🎬

SHOT (ショット・カット単位)

1 つのカット (連続する映像の最小単位)。カメラのカット〜カットの間。映像制作の基本作業単位。

  • 例: SHOT_001 (タイトルロゴ出現) / SHOT_023 (商品クローズアップ)
  • 1 SHOT = 複数 Task (工程) + 提出 asset (動画/画像) + QC 履歴 + コメント
  • 命名規則: SHOT_ + 3 桁数字 (例 SHOT_001)・Lighting/Composite/MattePaint/Animation/FX 等 全工程を内包
  • 各 SHOT に対し SHOT 関係者 thread (PM/Director/Lead/担当者) が QC/Review 提出時 自動作成される
⚙️

Task (工程・最小作業単位)

SHOT 内の各工程 (Lighting / Composite / MattePaint / Animation / FX 等)。担当者 1 名 + 期日 + 提出 asset (バージョン管理) + status (open/in_progress/reviewing/retake/approved) を持つ。

  • 例: Composite (Sato 担当) / Lighting (Kato 担当) / MattePaint
  • 1 Task = 1 担当者 + バージョン履歴 (v001 / v002 / ...) + QC ステータス
  • Review/QC 提出 → SHOT 関係者 thread に通知 → Director Approve → status=approved
  • status: 🔵 open (未着手) → 🟡 in_progress (作業中) → 🟠 reviewing (Director 判定待ち) → 🔴 retake (差し戻し) → 🟢 approved (承認済)

📐 階層イメージ (実例)

📁 marukome (Project)
└── 🎞️ SEQ001 (オープニング)
    └── 🎬 SHOT_001 (タイトルロゴ出現)
        ├── ⚙️ Composite (Sato 担当・v005 Approved)
        ├── ⚙️ Lighting (Kato 担当・v003 in_progress)
        └── ⚙️ MattePaint (Sato 担当・v002 reviewing)

  • • QC/Review 依頼通知 title: 📌 Review 依頼: marukome / SEQ001 / SHOT_001 / Composite
