🚪 退勤

当日 (current) / 前日忘れ分 (previous) 退勤報告の提出。ルーティン入力・進捗報告・問題報告も含む。

📸 画面

✨ 主な機能

  • 退勤時刻 — 自動で現在時刻 (修正可)。
  • 進行報告 — 本日完了したタスク・進捗率の入力。
  • 問題報告 (blocker) — 進行を妨げる課題 (技術トラブル / アセット待ち / 仕様確認等)。
  • 引継ぎ (handover) — 翌日対応してほしい事項を後任に明示。
  • 次優先 (next_priority) — 翌日の最優先タスクの宣言。
  • 前日忘れモード — mode=previous&return=routine で前日分を後追い提出。サイドメニュー非表示で 提出強制。

🛠 操作方法

  1. sidemenu 🚪 退勤 → 退勤報告画面。
  2. 各項目入力 → 送信 → routine 画面 or dashboard に遷移。
  3. 前日忘れの場合 ダッシュボード経由で 自動 prompt 表示。
