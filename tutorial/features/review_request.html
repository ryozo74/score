📌 Review 依頼

Compositor が Review 依頼を提出 → SHOT 関係者 thread (PM/Director/Lead + mention 主体) に自動投稿。Director の判定前段階で 部内意見集約。

📸 画面

✨ 主な機能

  • 提出種別 = Review — 部内レビュー目的・Director 判定前の意見集約
  • SHOT 関係者 thread 自動作成 — PM + Director + Lead + task assignees + mention で構成
  • mention 主体 — Director 自動補完なし。mention で送信先を 明示指定
  • 階層 title — proj / SEQ / SHOT / task の 4 階層を本文先頭に明記 (例: marukome / sq01 / c05 / comp)
  • QC ビューア link 自動付与 — 本文末尾に /qc/{shot_id}?task_id&asset_id 緑ボタン化
  • Push + SSE 配信 — 全 participants に通知 (sender は除外・但し mention 含 sender は配信)

🛠 操作方法

  1. /task/{task_id} で対象 asset card → 📤 既存 asset で QC 依頼 → modal で「📌 Review」選択
  2. mention 候補から PM / 他 user を選択 (必須・Director 不在時)
  3. 補足コメント (任意) → 📤 送信
  4. 受信側: msg → SHOT thread → ▶ QC ビューアで確認 click → 対象 asset 再生 → 部内コメント返信
