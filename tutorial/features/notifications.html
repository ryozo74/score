🔔 通知センター

全通知の集約画面。未読 / @ mention / お知らせ の 3 タブ。Calendar の DM thread が 🎬 SHOT (3+名) と 💬 DM (2 名) に区分表示。

📸 画面

✨ 主な機能

  • 未読 / @ mention / お知らせ タブ — タブで通知種別を絞り込み。
  • 🎬 SHOT thread / 💬 DM 区分表示 — Calendar の DM thread を 自動判別。
  • 📣 Push 通知を有効化 ボタン — ヘッダー右上。1 click で OS 通知 toast を有効化 (要 ブラウザ通知許可)。
  • 🧪 テスト送信 ボタン — Push + SSE 両方を発火して動作確認。OS toast + 画面右下 toast 両方表示。
  • 既読化 — 通知カード click で既読、sidemenu 💬 赤丸も連動して減算。

🛠 操作方法

  1. sidemenu 🔔 通知 → 通知センター。
  2. 📣 Push 通知を有効化 → ブラウザ通知許可 dialog → 「許可」 → 有効化。
  3. 🧪 テスト送信 → 結果 status line + 画面右下 toast + OS toast 確認。
  4. 通知 card click → 該当 thread or 画面へ遷移。
